This is an example project for the following article: [Using the Google Maps API with Vue.js](https://markus.oberlehner.net/blog/using-the-google-maps-api-with-vue/)

## Build Setup

```bash
# Install dependencies.
npm install

# Serve with hot reload.
npm run serve

# Build for production with minification.
npm run build
```
